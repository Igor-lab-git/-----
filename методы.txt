arr:
Splice()— это метод для изменения содержимого массива путём удаления или замены существующих элементов
REDUCE-(АККАМУЛЯТОР, элемент, индекс, массив)( свёртывает элементы массива в единое значение)
FIND-помогает найти первый элемент в массиве согласно переданному в параметре коллбэку.
PUSH()-добавляет неограниченное количество элементов в конец массива. При этом исходный массив изменяется, а результатом возвращается новая длина массива.
findIndex()- возвращает индекс первого найденного в массиве элемента, который подходит под условие переданной функции.
MAP-позволяет применить заданную функцию для каждого элемента массива. Метод в параметре получает функцию, которая выполнится для каждого элемента массива
COUNT-функция, которая позволяет разработчикам быстро и легко подсчитать количество вхождений определённой сущности или переменной
pow()-объекта Math,возводит число в степень.let a=Math.pow(3,2)//9
Math.abs(-3); // 3 Выведите на экран абсолютное значение числа (положительное число без минуса)
Math.ceil() всегда округляет число в большую сторону

textContent- позволяет получить и изменить текст элемента.
getItem-предназначен для получения данных. Он принимает один параметр - ключ, под которым эти данные были сохранены.
remove()-позволяет удалить элемент. Применяется к тому элементу, который нужно удалить.
